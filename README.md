## install modules
```
npm install
```

## 1.1) to run tests for mobile browser - run server  
```
appium --allow-insecure chromedriver_autodownload
```

## 1.2) then after run server:
```
npm run mobile
```
## 2.1) to run tests for mobile client - run server
```
appium
```

## 2.2) then after run server:
```
npm test
```
